परिचय

Ye guide aapko step-by-step batata hai kaise 5yberBind style website ko GitHub Pages ya Netlify par deploy karein.

Steps

  1. Create a new GitHub repo and push files
  2. Enable GitHub Pages from repo settings (branch: gh-pages or main)
  3. Or use Netlify: drag & drop the site folder
  4. Replace assets and downloads with real files
